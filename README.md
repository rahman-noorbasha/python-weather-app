# Python Weather App

A Python command-line weather application that fetches live weather data using the Open-Meteo API.

## Features

- Search weather by city name
- Converts city name into coordinates
- Fetches live weather data
- Displays:
  - Temperature
  - Wind Speed
- Handles invalid city names

## Technologies Used

- Python
- Requests Library
- Open-Meteo API

## How to Run

```bash
python weather_app.py
